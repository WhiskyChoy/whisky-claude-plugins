# whisky-claude-plugins

Personal Claude Code plugin marketplace.

## Installation

```bash
claude plugin marketplace add https://github.com/whisky/whisky-claude-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| claude-statusline | Custom terminal statusline for Claude Code sessions |

## Installing a Plugin

```bash
claude plugin install claude-statusline@whisky-claude-plugins
```
